FlexBox

Documentação

Display: flex

A primeira propriedade que precisa ser declarada para que o Flexbox funcione!

Algumas propriedades como flex-direction, justify-content e align-content já vem com os valores padrão.

Propriedade Valor
flex-direction row
justify-content flex-start
align-content flex-start

Exemplo:

Justify-content

A justify-content é a propriedade que alinha os itens do container flexível quando os itens não ocupam todo o espaço disponível no eixo principal (horizontalmente).

Exemplos:



flex-end

Alinha os itens no final do contêiner ao longo do eixo principal. Os itens terminam no final do contêiner, e pode haver espaço vazio no início.

flex-start

Alinha os itens no início do contêiner ao longo do eixo principal. Os itens começam a partir do início do contêiner e podem haver espaço vazio no final, dependendo do tamanho do contêiner.

center

Alinha os itens no centro do contêiner ao longo do eixo principal.Os itens são distribuídos de forma igual em ambos os lados do centro.

space-between

Distribui os itens de modo que o espaço entre eles seja igual, com o primeiro item no início do contêiner e o último item no final do contêiner. Não há espaço vazio no início ou no final do contêiner.

space-around

Distribui os itens com espaços iguais em torno deles.Isso significa que há espaço vazio igual tanto no início quanto no final do contêiner, bem como entre os itens adjacentes.

space-evenly

Distribui os itens de modo que o espaço entre eles e em torno deles seja igual.O espaço vazio no início e no final do contêiner é igual ao espaço entre os itens adjacentes.

stretch

Estica os itens para preencher todo o espaço disponível ao longo do eixo principal.Isso é usado principalmente quando você deseja que os itens ocupem todo o espaço disponível.

flex-direction

A propriedade CSS flex-direction é usada para definir a direção principal em que os itens flexíveis são dispostos dentro de um contêiner flexível. Ela determina se os itens são dispostos horizontalmente ou verticalmente e se a ordem deles deve ser revertida.

Exemplos:

row

Esse é o valor padrão. Os itens são dispostos na direção horizontal, da esquerda para a direita. O eixo principal é horizontal.

align-items

A propriedade CSS align-items é usada para alinhar os itens flexíveis (ou conteúdo) dentro de um contêiner flexível ao longo do eixo transversal. O eixo transversal é perpendicular ao eixo principal do contêiner

Exemplos:

stretch

Este é o valor padrão. Os itens são esticados para preencher todo o espaço disponível ao longo do eixo transversal. Isso é útil quando você deseja que os itens tenham a mesma altura (verticalmente) dentro do contêiner.

flex-start

Alinha os itens no início do contêiner ao longo do eixo transversal. Os itens começam a partir do início do contêiner no eixo transversal.

flex-end

Alinha os itens no final do contêiner ao longo do eixo transversal. Os itens terminam no final do contêiner no eixo transversal.

center

Alinha os itens no centro do contêiner ao longo do eixo transversal. Os itens são centralizados verticalmente dentro do contêiner.

baseline

Alinha os itens com base na linha de base de seu conteúdo. Isso é útil quando você tem texto em diferentes itens e deseja alinhar seus textos pela linha de base.

flex-wrap

A propriedade CSS flex-wrap é usada para controlar como os itens flexíveis são dispostos dentro de um contêiner flexível quando não há espaço suficiente ao longo do eixo principal para acomodar todos os itens. Ela determina se os itens flexíveis devem ser quebrados em várias linhas ou se devem ser ajustados de alguma outra forma.

Exemplos:

nowrap

Os itens são forçados a permanecer em uma única linha, mesmo se o espaço não for suficiente para acomodar todos eles.Não há quebras de linha automáticas.
